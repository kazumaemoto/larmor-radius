# larmor-radius

## About

larmor-radius calculates larmor radius of particles.
Particles (listed below), temperature as electron volts, and magnetic field as tesla are required.

## Requirements

- python
- numpy
- scipy
- matplotlib

## Usage

```
$ python larmor-radius.py
Particles: 
Temperature (eV): 
Magnetic field (T):
```

## Particles

- e: electron
- Ar: Argon
- Kr: Krypton
- Xe: Xenon
